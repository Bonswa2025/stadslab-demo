# Stadslab Demo (Backoffice + Manuals)
Een minimale Next.js app die jouw twee modules mount:
- /backoffice → Stadslab Backoffice
- /manuals → Foodtruck Manuals

## Development
npm install
npm run dev

## Production build
npm run build
npm start

## Deploy op Vercel
- Maak een GitHub-repository en upload deze map.
- Koppel de repo in Vercel (New Project → Import).
- Framework preset: Next.js (autodetect).
- Build command: `next build` (automatisch).
